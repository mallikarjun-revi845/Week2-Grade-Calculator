# Student Grade Calculator

## Project Overview

This project calculates a student's grade based on marks entered by the user.

## Objectives

* Learn if-elif-else statements
* Learn functions
* Learn loops
* Learn input validation
* Learn error handling

## Grading Logic

A : 90-100

B : 80-89

C : 70-79

D : 60-69

F : 0-59

## Features

* Student name input
* Marks validation
* Grade calculation
* Encouraging messages
* Function-based implementation

## How to Run

python grade_calculator.py

## Concepts Used

* Functions
* While Loop
* If-Elif-Else
* Exception Handling
* User Input

## Learning Outcomes

This project helped me understand decision-making and loops in Python.
